💡 首次使用配置指南

1. 获取连接信息

在 Supabase 控制台 → Project Settings → API 中复制：

• URL: Project URL

• Anon Key: anon public（客户端使用的公开密钥）

2. 创建数据表

在 SQL 编辑器中执行：

DROP TABLE IF EXISTS stocks CASCADE; CREATE TABLE stocks ( id SERIAL PRIMARY KEY, stock_name VARCHAR(100) NOT NULL, stock_code VARCHAR(20) NOT NULL UNIQUE, market VARCHAR(50) NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); NOTIFY pgrst, 'reload schema';
